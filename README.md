# 🦺 EPI-Control - Sistema de Gerenciamento de EPIs

O EPI-Control é um sistema backend desenvolvido em Java com Struts 2 para o gerenciamento de Equipamentos de Proteção Individual (EPIs) em organizações.
A aplicação permite o cadastro, edição, exclusão e busca de EPIs, facilitando o controle e a rastreabilidade desses itens essenciais para a segurança do trabalho.

## 🚀 Funcionalidades

✅ Cadastro de novos EPIs (nome, categoria, descrição, etc.)

🔍 Busca de EPIs por nome

✏️ Edição de informações de EPIs cadastrados

❌ Exclusão de EPIs do sistema

📋 Listagem geral de EPIs registrados

## 🛠️ Tecnologias Utilizadas

- [Java 8](https://www.oracle.com/br/java/)
- [Struts 2](https://struts.apache.org/)
- [MySQL](https://www.mysql.com/)
